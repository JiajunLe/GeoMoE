# GeoMoE implementation

Pytorch implementation of GeoMoE for AAAI'26 paper ["Divide-and-Conquer Motion Field Modeling with Mixture-of-Experts for Two-View Geometry"](https://arxiv.org/abs/2508.00592), by [Jiajun Le](https://scholar.google.com/citations?user=uWhzrG4AAAAJ&hl=zh-CN&oi=sra) and [Jiayi Ma](https://scholar.google.com/citations?user=73trMQkAAAAJ&hl).
The code will be made publicly available soon.
